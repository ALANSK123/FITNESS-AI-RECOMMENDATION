# FITNESS-AI-RECOMMENDATION
AI-powered backend for personalized fitness recommendations. Built with Spring Boot, Spring Data JPA, and PostgreSQL

This is a Spring Boot project built for practice and learning.
This project was built by following Spring Boot Full Stack AI Microservices Project Course. (https://youtu.be/nJyfLqFpbp4?si=pr-RNkgJQb76nlRx)

## Features
- REST API with Spring Boot
- Basic project structure
- Backend development practice

## Tech Stack
- Java
- Spring Boot
- Maven

## How to Run
1. Clone the repository
2. Open the project in your IDE
3. Run the application
4. Open the app on your local server

This project was created as part of a learning/tutorial project.
